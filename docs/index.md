# SMSTU MDS Dissertation LaTeX Template

**Author:** Arpan Sapkota  
**Year:** 2025  

This repository provides a LaTeX dissertation template for
Master's in  Data Science (MDS) students under IOST SMSTU guidelines.

## Abstract
This template standardizes academic dissertation formatting using LaTeX
and Overleaf, enabling students to focus on research content rather than layout.
