# Contributing Guidelines

Thank you for your interest in contributing to this project.

## Ways to Contribute
- Fix formatting or compilation issues
- Improve documentation
- Add new chapter templates
- Improve Overleaf compatibility
- Report bugs or suggestions via Issues

## Contribution Rules
1. Fork the repository
2. Create a new branch for your changes
3. Ensure the template compiles successfully
4. Submit a Pull Request with a clear description

## Academic Ethics
- Do NOT remove original author attribution
- Follow academic integrity standards

All contributions are reviewed before merging.
